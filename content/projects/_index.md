# Software

- [Pluto](https://github.com/thestaccato/pluto) - A minimal and suckless window manager for [river](https://codeberg.org/river/river). 
- [Krypton](https://github.com/thestaccato/krypton) - A CLI tool and a rust library for creating encrypted vaults or encrypting single files with metadata protection. 
- [Hyprwal](https://github.com/thestaccato/hyprwal) - A wallpaper utility for Wayland compositors that uses wl_surface/wl_shm for standalone background surfaces.
- [Mozart](https://github.com/thestaccato/mozart) - A TUI music player for those who spend their most life in terminal.

- [SFM](https://github.com/thestaccato/sfm) - As per the name suckless's style terminal file manager.

- [Dendrites](https://github.com/thestaccato/dendrites) - A ML model based on random forest classification for personalised medication.
